# Personal Budget Tracker (Microsoft Excel)

Automated budget workbook to monitor income and expenses.

## Contents

- **Personal_Budget_Tracker.xlsx** — open in Microsoft Excel or Google Sheets (File → Import).

## Features

- Track income and expenses with **SUM**, **IF**, and **AVERAGE**.
- Dashboard KPIs, budget vs actual, and charts for category spending.

## How to use

1. Clone or download this repository.
2. Open `Personal_Budget_Tracker.xlsx` locally in Excel, or upload/import it in Google Sheets.

GitHub stores the file for download and version history; calculations run in Excel or Sheets after you open the workbook.
